The Face We Make tutorial

Hi! This is a research application that wants to understand how good are humans identifying feelings using emoticons.

The application is really simple. It loads a photo from The Face We Make's photostream and asks you this question: What emoticon is he/she representing?

You will have to choose between the following possible answers:

Click in the emoticon that you think the person is representing and a new task will be loaded for you. That's all!

We will show you 10 photos, and if you do all of them in a row we will show you a summary about which ones you have answered correctly. Something like this:
